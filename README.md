## ğŸš€ Getting Started

<details>
<summary><b>Installation & Development</b></summary>

### Prerequisites
- Node.js (Latest LTS)
- npm, pnpm, or yarn

### Setup
```bash
# Clone the repository
git clone https://github.com/z1921531571/imas-lyrics-visualizer.git

# Install dependencies
npm install
```

### Development
```bash
# Start the development server
npm run dev
```

### Build & Search Indexing
Since Pagefind indexes static files, you need to build the project to update the search index:
```bash
# Build the project
npm run build

# Preview the build locally
npm run preview
```
